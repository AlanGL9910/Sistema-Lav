"# Sistema-Lav" 
“Sistema de Gestión de Órdenes y Autenticación por QR para Negocios de Entrega Diferida”

Este sistema es una plataforma diseñada para comercios donde el cliente deja un producto para recibirlo posteriormente, como servicios técnicos, reparaciones, mantenimiento o procesos especializados. Su objetivo es agilizar la recepción, el cobro y la autenticación de la persona que recoge el producto, aumentando rapidez, organización y seguridad.
